                Installing Git

If on windows just download it and install like any other app.
On Linux install through it via terminal,
For debian: sudo apt-get install git 



Setting up SSH key windows

1. open CMD
2. paste this command and change email: ssh-keygen -t ed25519 -C "your_email@example.com"
3. Press enter couple of times to accept the default save location
4. Then paste this command and change the PATH: type %userprofile%\.ssh\id_ed25519.pub | clip
5. Now u have copied the shh to clipboard
6. Open GitHub 
7. Go to settings 
8. Open SSH and GPG keys section
9. Click New SSH key
10. Paste the SSH key to the field 
11. Click save and its done

Making a repo and pushing


Create a repo in browser and copy this 

    Open git in the folder with files to commit 
run these commands: 
1. git init         -> initialize git
2. git add .        -> adding all files to local repo
3. git status       -> showing all files to be committed 
4. git commit -m "first commit"  -> adding first commit to local repo
5. git remote add origin https://github.com/Current repo from that screen
6. git push -u origin master      -> pushing changes to GitHub
