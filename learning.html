WORK IN PROGRESS

I'm currently working on building my networking knowledge from the ground up, as I've become a bit rusty since leaving school.
I'm also teaching myself web development, and I'm learning Git in the process. Although I find web development a bit boring at times, I recognize that it's a valuable skill to have even if I don't plan to make websites.

In addition to these topics, I'm learning ethical hacking on Hack the Box by working my way through the Starting Point machines. I've made it to Tier 2, which is the last one.
While I'm enjoying learning about hacking, I'm actually more interested in blue teaming for the time being. I guess we'll see where my interests take me as I continue to learn!
